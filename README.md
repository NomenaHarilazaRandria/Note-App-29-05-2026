# Note-App-29-05-2026
A simple and responsive Notes App built with HTML, CSS and vanilla JavaScript. Users can create, edit, delete and store notes locally using browser localStorage.


## Features

- Add new notes
- Edit existing notes
- Delete notes
- Character counter for textarea
- Automatic saving with localStorage
- Responsive design for mobile devices
- Notes displayed with creation date

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage API

## Project Structure

├── index.html  
├── style.css  
└── script.js  

## How It Works

### Add a Note
Users can enter:
- a title
- note content

Then click the **"Ajouter La Note"** button to save it.

### Edit a Note
Click the **"Modifier"** button to load the selected note into the form and update it.

### Delete a Note
Click the **"Supprimer"** button to remove a note permanently.

### Data Persistence
All notes are saved in the browser using `localStorage`, so they remain available after refreshing the page.
